# AI Healthcare Website

An AI-powered healthcare information platform with interactive features including body mapping, symptom checker chatbot, and cancer awareness articles.

## Features
- 🏥 Interactive Body Map - Click on body parts for health information
- 🤖 AI Chatbot - Get instant responses about common symptoms
- 📚 Cancer Articles - Comprehensive information about various cancers
- 📱 Fully Responsive - Works on all devices

## Technologies Used
- HTML5
- CSS3 (with animations and flexbox/grid)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts

## How to Use
1. Clone or download this repository
2. Open index.html in your web browser
3. Navigate through different sections
4. Try the interactive features!

This website is for educational purposes only and should not replace professional medical advice. Always consult with healthcare professionals for medical concerns.
